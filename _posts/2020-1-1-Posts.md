- **VIRTUAL NETWORK SECURITY FUNCTIONS (VNSFS)**: vNSFs are instantiated within the network infrastructure by a vNSF orchestrator in order to effectively monitor and filter network traffic in a distributed manner. 
- **DATA ANALYSIS AND REMEDIATION ENGINE (DARE)**: an information-driven IDPS platform capable ofpredicting specific vulnerabilities and attacks by relying on Big Data, Threat Monitoring and Machine Learning to analyse the output produced by vNSFs.
- **TRUSTED INFRASTRUCTURE**: The infrastructure attestation binds the vNSFs and the network configuration with the store and orchestration of the network
